ClauLi · Pixl · Blob
один экран, три героя

Селектор героя сверху меняет только отрисовку. Все настройки — лайаут, цвет, часы, текст, тайминги — остаются на месте и применяются к любому герою. Анимации полностью переработаны: всегда живой «слой жизни» (дыхание, микро-саккады, естественное моргание) + сигнатура движения под каждую эмоцию + своя пластика у каждого героя. Это макет для ревью — пока ничего в прошивке не меняется.

02 · эмоции · ClauLi

Живые эмоции героя

Каждая карточка анимируется постоянно: слой жизни + сигнатура эмоции + вход (squash-pop). Переключи героя — сетка перерисуется его языком формы и пластикой.

Дизайн вдохновлён EMO (Living.AI): только идея выразительности, ассеты не копировались. Все герои (ClauLi/Pixl/Blob) и схемы — собственные оригинальные иллюстрации. Рендер повторяет логику прошивки (залитый силуэт + bloom + блик + веки) и веб-превью (drawTileEye).
